  1. Instalação do framework DJANGO

    1. Criei uma pasta (experimentos) para o projeto
    2. Iniciei um ambiente virtual para isolar as dependências necessárias
      3. Comandos usados no processo:
      • python -m venv venv
      • venv/scripts/activate
    3. Instalei o Django
      4. Comandos usados no processo:
      • pip install django
    4. Criei o documento requirements.txt
      5. Comandos usados no processo:
      • pip freeze > requirements.txt

  2. Esqueleto base do Projeto

    1. Iniciei o projeto com o nome praticando
      2. Comandos usados no processo:
      • django-admin startproject praticando .
    2. Iniciei o app pagina e etapas

      3. O objetivo é ter uma home com um menu de rotas aninhadas para cada parte do processo, e o app etapas deve gerenciar essas etapas (arquivo HTML como este).

      Comandos usados no processo:
      • python manage.py startapp pagina
      • python manage.py startapp etapas
