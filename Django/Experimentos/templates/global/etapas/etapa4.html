  1. Criando um menu de rotas dinâmicas

    1. Criando uma pasta base no template/global
      • Escrever código HTML para o menu de etapas.
      • Usar a tag include do Django.

        • Isso permite incluir o menu em todas as rotas das aplicações.
        Também possibilita adicionar um cabeçalho com formato padrão para todos os arquivos, evitando repetição de código.

    2. Padronizando o projeto
      • Criar um arquivo para o cabeçalho padrão.
      • Adicionar um contexto como terceiro parâmetro em todas as views.

        • Dentro do dicionário de contexto, adicionar apenas uma chave chamada "titulo" e definir seu valor como uma string que indica a etapa atual do projeto exibida.

    3. Formatando o cabeçalho
      • Configurar o idioma para pt-br.
      • Configurar o título da página usando a variável "titulo" dentro do contexto individual de cada view gerada.
      • Incluir o menu de navegação no cabeçalho.
    4. Adicionando o cabeçalho padrão
      • Incluir o cabeçalho em todos os arquivos.

        • Adicionar dentro da tag include do Django o caminho do arquivo → global/base/cabecalho.html.

      • Definir o título usando a tag de variável do Django chamando a variável titulo definida no contexto de cada view.

        • Isso permite alterar o título de cada página de forma dinâmica. Em qualquer parte do HTML, temos o head exibindo o título na parte superior da janela e também podemos definir a tag h1 com o mesmo nome.
