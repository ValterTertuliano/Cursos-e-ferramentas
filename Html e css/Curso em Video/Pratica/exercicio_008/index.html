Principais formatações

Negrito para destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semântica)

Podemos usar a tag <strong> para destacar em negrito.

Itálico para dar ênfase

Nesta frase temos um termo em itálico usando a tag <i> (não semântica)

Podemos usar a tag <em> para dar ênfase com itálico.

Texto marcado

Podemos criar texto marcado usando a tag <mark>

Dentro do HTML não podemos mudar a cor diretamente, precisamos do CSS.

O que podemos fazer é adicionar um parâmetro dentro da tag, chamado style. Dentro desse parâmetro, já podemos considerar o CSS <mark style='background-color: nomeDaCor;'>

Também devemos lembrar que adicionar o parâmetro style dentro da tag modifica apenas o que está dentro da tag, e não as outras -> observe que a marca está igual à primeira.

Podemos resolver isso.

Dentro de head podemos criar uma tag <style> e criamos um seletor para mark, mudando a cor de todas as marcas dentro do HTML em que está escrito.

Observe que a única marca que não se modifica para a cor aqua foi a tag com o parâmetro adicionado manualmente

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo. Usamos a tag <big> e a <small>, observe que a tag <big> está obsoleta, enquanto a tag <small> continua em uso.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois.

Existe também o sublinhado com a tag <u> (não semântica).

Texto sobrescrito

Usamos a tag <sup> para coisas como x10+3. Com essa tag podemos colocar esse tipo de equação de forma adequada, por exemplo: x10+3

Texto subscrito

Da mesma forma, podemos usar a tag <sub> para coisas como H2O, que é o oposto da tag <sup>, deixando esse tipo de fórmula mais adequada ao seu contexto, por exemplo: H2O
