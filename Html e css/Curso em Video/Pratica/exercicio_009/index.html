codigo-fonte

O comando document.getElements é escrito em javascript, o problema disso é que as letras podem ficar muito juntas, confundindo a leitura

Podemos usar a tag <code> para resolver isso, deixando a fonte monoespaçada

exemplo: document.getElements("teste")


Podemos usar o code para escrever codigos de exemplo em paginas

for x in range(0, 100, 2): print(x)

mas apenas adicionar a tag <code> não vai deixar isso com o visual adequado, como acabamos de ver

para resolver isso podemos usar a tag <pre> que permite deixar o texto com a mesma formatação que está no html, isso é bem útil

exemplo: 

            
    for x in range(0, 100, 2):
        print(x)
isso com certeza ajuda muito
